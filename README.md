# Tasca S1.06: Advanced PHP

This repository contains the resolution of the exercises for the "Advanced PHP" module. The focus of these exercises is to practice superglobal variables, sessions, magic methods and Traits.

## 🚀 Level 1: Basics of Advanced PHP

### Exercise 1: Forms & Sessions
- **Description**: Creation of an HTML form that sends data to a PHP script.
- **Goals**: 
    - Retrieve form data using `$_POST` or `$_GET`.
    - Store and display information using the `$_SESSION` superglobal.

### Exercise 2: Magic Constants
- **Description**: Implement a program that utilizes at least two PHP Magic Constants (e.g., `__DIR__`, `__FILE__`, `__LINE__`, etc.).

### Exercise 3: Magic Methods
- **Description**: Override the logic of at least one PHP Magic Method (excluding `__construct`). Common choices include `__toString()`, `__get()`, or `__set()`.
